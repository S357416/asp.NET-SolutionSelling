Solution Selling
This project aims to meet client requirements of an ecommerce website that will allow users to put items up for sale. Other users can then view these items and purchase them. 

Installation
Open Nu Package Manager

Migrate if needed (included migrated file)
Run 'add-migration InitialCreate'
Run 'Update-Database'

Usage Instruction
The migrated database is seeded with an admin user, admin role and 5 example items.
